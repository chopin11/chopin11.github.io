思考

如果通过如下命令进行测试访问，curl 命令行是会报错的

你可能会有疑问，之前测试 HTTP 不就可以的吗？

解答如下：

1. 服务器证书校验是在 TLS 握手期间进行的，这时还没有解析到 HTTP 层面 Host 字段，TLS 是看不到 -H 指定的域名

2. --resolve 选项是模拟 DNS 解析的缓存，会把域名信息通过 SNI 字段传递给服务器的；
